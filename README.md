	- download link : 

	for windows:
	https://www.lfd.uci.edu/~gohlke/pythonlibs/#opencv

			opencv_python‑3.3.1‑cp35‑cp35m‑win_amd64.whl

			'pip opencv_python‑3.3.1‑cp35‑cp35m‑win_amd64.whl'

        for Mac or Linux:
		- conda command
